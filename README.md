# ✈️ Travel Cost Prediction using Machine Learning
## 📌 Project Overview

This project predicts travel expenses based on factors like Distance_km, transport type, number of days, number of people, and hotel_rating.
It uses Machine Learning algorithms to estimate trip costs accurately, helping users plan budgets efficiently.

## 🚀 Features
1. Predict travel cost instantly
2. User-friendly input parameters
3. Supports multiple travel factors
4. Accurate ML model predictions

## 📊 Dataset

The dataset contains the following features:

1. Distance_km
2. Transport ( Bus, Train, Flight  )
3. Days
4. People
5. Hotel_rating
6. Total Cost (Target Variable)

7. ##🧠 Machine Learning Model
Algorithm Used:  Linear Regression

1. Data Preprocessing 
2. Handling missing values
3. Encoding categorical data
4. 4. Model Evaluation:  R² Score
  
## 🛠️ Tech Stack
1. Python 🐍
2. Pandas & NumPy
3. Scikit-learn
4. Matplotlib / Seaborn
5 Jupyter Notebook

##📈 Example Input
Distance_km: 500
Transport: Flight(3)
Days: 2
People: 2
Hotel_rating: 3

Output:
👉 Estimated Travel Cost: 8556
